# Jurojin — brand & business context

Single source of truth for **who Jurojin is and how we talk**, loaded by the

## What Jurojin is

Jurojin (JurojinPoker, `jurojinpoker.com`) is a **freemium desktop software for
online poker players**, built to make **multi-tabling** easier, faster and more
comfortable. Positioning statement we sustain:

> **The #1 multi-table & comfort software for online poker.**

Frame Jurojin as a **transversal tool — the player's true companion during the
session.** It's the layer that unifies organization, hotkeys, bet sizes and
real-time comfort/decision aids so the player can focus on playing.

**Gaming, not gambling.** A core brand stance: we pull poker toward the
*gaming* world (skill, tools, setup, flow) and away from the *gambling* framing.
Tone and word choices should reflect that.

### Not a tracker — and that's fine

Jurojin is **not a poker tracker** and does **not** mine hand histories for
opponents' stats (PokerTracker / Hand2Note / HM2 territory). Its overlays are
**real-time, hero-focused decision aids**. Important: trackers are **not
competitors** — they're **complementary**, and many players use both. Don't
position against them. By default we **don't name any allied or competing brand**;
only mention another tool if the specific piece is explicitly about it.

## The problem it solves

Playing many tables at once splits focus, fades attention and causes input
errors. Jurojin reduces that cognitive load so players can **raise their table
count and hands/hour volume** comfortably, in a "flow state", with fewer mistakes.

## Who we talk to (audience)

A **broad** audience — all profiles of poker lovers — with different entry points:

- **Recreational / aspirational players:** see Jurojin as the start of
  *solidifying their session practices* — standardizing bet sizes and having
  data on hand to apply what they're learning (Pot Odds, SPR, MDF).
- **Experienced players & real pros:** at minimum use the **organization**
  capabilities to maximize focus and minimize distractions, and to **unify basic
  functions** (hotkeys, bet sizes) across rooms.
- **Anyone leveling up multi-tabling:** benefits from **peripheral comfort
  features** — auto "sit back in" (avoiding sit-outs), font increase, stack
  coloring, urgency marking / colored borders on tables that need action, etc.

Tone is broadly inclusive across levels — speak to the player's goal, not their
rank. The product is **integral and accessible for every poker profile.**

**Languages:** the app ships in **9 languages** — Spanish, English, Russian,
Chinese, Japanese, Italian, French, Portuguese, German. Suitable for all
audiences; no single region dominates.

## What we offer (modules)

Jurojin is organized into feature **modules**, with paid **bundles** on top of a
free tier:

- **Hotkeys** — keyboard/mouse shortcuts for poker actions: Fold, Check/Call,
  Bet/Raise, All-In, preset Bet Sizes, increment/decrement by SB/BB, sit-out
  options (incl. auto sit-back-in), table-focus moves (give focus to most urgent
  table, cycle stacked tables, full-keyboard mode).
- **Overlays / HUDs** — real-time decision overlays: Pot Odds, Effective Stack,
  SPR (+ next-street prediction), MDF / Bluff Equity / Min Outs, RNG roll,
  **Actions Line / actions history by street**, **BetSizes GamePad** (up to 16
  sizes in BB, % pot, multiplier, geometric), Player Positions, Tables Counter.
- **Tiling / Table Management** — arrange tables (basic grid, stacked, mixed).
  **Dynamic tiling** auto-moves important tables into focus:
  - *Action Required (AR)* — table jumps to an "AR slot" when hero must act,
    then returns (with/without stacking).
  - *In-Hand* — table stays in focus for the whole hand.
  - *Stack & Tile / VPIP-based* — table moves when you put money in the pot.
- **Comfort / quality-of-life** — font increase, stack coloring, urgency
  borders/marking, focus-management aids.
- **Site-specific** — e.g. a **Bodog** bundle (auto top-up, convert blinds).

## Glossary (canonical terms)

| Term | Meaning |
|------|---------|
| Multi-tabling | Playing several tables at once — the core use case |
| Tiling | Arranging tables on screen (grid / stacked / mixed) |
| Stacking | Tables overlapped in one spot, one visible at a time |
| Dynamic tiling / layout | Tables auto-move on a trigger (AR / In-Hand / VPIP) |
| Action Required (AR) slot | Screen area a table jumps to when hero must act |
| In-Hand (dynamic) | Table stays focused for the whole hand |
| VPIP | Voluntarily Put money In Pot — a Stack & Tile trigger |
| Layout | A saved table-arrangement profile |
| Slot | A position in a layout (Play slot, AR slot) |
| Overlay / HUD | Real-time widget on top of a table (Heads-Up Display) |
| BetSizes GamePad | Overlay/keypad of preset bet sizes |
| Hotkeys | Configurable keyboard/mouse shortcuts for actions |
| Hero / Villain | The player / an opponent |
| Freemium | Free to use, with paid feature bundles |
| Room / poker site | A supported online poker platform |

## Voice & tone

- **Person:** second person ("you"), across 9 languages.
- **Personality:** **jovial, approachable to everyone**, player-to-player,
  practical, community-driven (Discord). We lean into the **gaming** feel of
  poker (setup, tools, flow, skill) over a gambling tone.
- **Do:** lead with the player benefit (comfort, focus, more tables, fewer
  errors), then name the feature; be concrete; speak to the goal, not the rank.
- **Don't:** call Jurojin a tracker; position against trackers; overpromise
  winnings; gambling-hype clichés; internal/architecture framing; invent features
  or unsupported rooms; quote prices.

## Background for the writer (context, not necessarily copy)

JurojinPoker is a **third-party developer tool.** It stays in contact with
casinos' security teams and operates **in compliance with each room's ToS** —
which is why feature availability differs per room. However:

- We are **not** official partners and **not** a casino/poker room.
- We **cannot guarantee the security of a user's account.**

This is **context to keep the writing accurate and safe** — it does **not** all
need to appear in a piece. Don't imply partnership/endorsement by a room, and
don't make account-safety guarantees.
